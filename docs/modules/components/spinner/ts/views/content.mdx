# Spinner

El componente Spinner se utiliza para indicar que una operación está en curso y que el usuario debe esperar. Este
elemento de la interfaz de usuario es especialmente útil durante la carga de datos o la realización de tareas que
requieren un tiempo considerable para completarse.
