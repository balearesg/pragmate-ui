# Modal Confirm

El componente `Modal Confirm` se utiliza para mostrar un mensaje de confirmación a los usuarios antes de realizar
acciones críticas en una aplicación web, como eliminar o modificar datos. El componente contiene dos botones: Aceptar y
Cerrar. Al hacer clic en el botón Aceptar, el usuario confirma la acción, mientras que al hacer clic en el botón Cerrar,
simplemente cierra el modal sin realizar ninguna acción.
