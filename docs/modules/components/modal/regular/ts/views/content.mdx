# Modal Regular

El componente `Modal Regular` se utiliza para mostrar información, alertas o contenido adicional en una aplicación web o
de escritorio. El componente modal es un elemento de la interfaz de usuario que aparece encima de otros elementos y
bloquea la interacción con el contenido subyacente hasta que se cierre. Funcionalidades Muestra contenido de manera
modal en la aplicación. Puede contener texto, imágenes, botones, enlaces y otros elementos. Puede ser personalizado con
estilos y temas propios. Se puede manejar de forma dinámica, mostrando u ocultando según sea necesario.
