# Code

El componente `Code` tiene como objetivo mostrar fragmentos de código que los usuarios pueden copiar y pegar. Este
componente es útil para aplicaciones o sitios web que necesitan mostrar ejemplos de código, explicaciones técnicas o
ayuda para el usuario. Puedes ver este componente en acción a lo largo de toda esta documentación.

## Implementation :

<Code>{`import { Code } from 'pragmate-ui/code';`}</Code>

<Code>{`<Code>const bar = 1;</Code>`}</Code>

Puede implementar su codigo ya sea agregando la expresion adentro de llaves o si desea agregarla como cualquier texto
que quiere tener ahi mismo puede hacerlo agregando las comillas inversas adentro de las llaves

## Agregando una constante :

<Code>{implementation}</Code>

<Code>{` const bar = 1 ; `}</Code>
```
