# Alert

El componente `Alert` está diseñado para mostrar mensajes de alerta en tu aplicación. Es un elemento de interfaz de
usuario crucial para informar a los usuarios acerca de eventos o cambios críticos. Este componente soporta cuatro tipos
de alertas: `Success`, `Error`, `Warning` e `Info`.

## Implementacion

Para incluir el componente `Alert` en tu proyecto, puedes usar la siguiente declaración de importación:

<Code>{`import { Alert } from 'pragmate-ui/alert';`}</Code>

## Uso

Para utilizar el componente Alert, debes pasar el atributo `type` para especificar el estilo de la alerta y children
para mostrar el contenido. A continuación se muestra la implementación básica:

<Code>{implementation}</Code>

# Propiedades

## Propiedades Obligatorias

-   `type`: Especifica el tipo de alerta. Puede ser uno de los siguientes valores:

        - `Success`
        - `Error`
        - `Warning`
        - `Info`

-   `children`: Contenido que se mostrará dentro de la alerta.

## Propiedades Opcionales

-   `message`: (string) Permite el manejo de mensajes de forma directa.
-   `closable`: (boolean) Determina si la alerta puede cerrarse con un botón.
-   `onClose`: (función) Función que se ejecutará cuando la alerta se cierre.
-   `icon`: (boolean) Opción para agregar íconos correspondientes a cada tipo de alerta.
