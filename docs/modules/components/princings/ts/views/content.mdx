# Pricing

El componente Pricing es un componente de React diseñado para presentar distintos planes de precios de manera efectiva
en tu aplicación o página web. Puedes personalizar cada plan mediante diversas propiedades (props). El componente
`Pricing` es un componente de React que permite mostrar diferentes planes de precios en una página web o aplicación.
Este componente acepta varias props que permiten personalizar el título, el precio, la descripción y las funciones
adicionales de cada plan de precio.

## Propiedades

-   variant: ("free" | "premium"): Define la variante del plan de precios que se desea mostrar.
-   title: (string): Establece el título del plan de precios.
-   description: (string): Añade una descripción al plan de precios.
-   price: (number): Requerida solo para la variante "premium". Indica el precio del plan.
-   advantages: (array): Una lista de las ventajas o características del plan de precios.
