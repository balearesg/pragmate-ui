# Progress Bar

Es utilizado para mostrar de manera visual el progreso de una tarea en particular. Es especialmente útil para dar
retroalimentación en tiempo real a los usuarios sobre tareas en curso o por completar.

## Implementacion

<Code>{`import { ProgressBar } from 'pragmate-ui/progress-bar';`}</Code>

<Code>{`<ProgressBar value={50} maxValue={100} variant="primary" />`}</Code>

## Propiedades

-   `value` (número): el valor actual de la barra de progreso.
-   `maxValue` (número, opcional): el valor máximo de la barra de progreso. Por defecto es 100.
-   `variant` (string, opcional): la variante de color de la barra de progreso. Por defecto es "primary".
