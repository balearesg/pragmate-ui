# Input

The `Input` component is a versatile React component designed for creating input fields within forms. It supports
multiple variants, types, and configurations, making it adaptable for various user interface designs.

### Import

<Code>{`import { Input } from 'pragmate-ui/form';`}</Code>

Code:

<Code>{`<Input type="text" name="name" label="Insert a text..." onChange={handleChange} value={value} />`}</Code>

#### Properties:

-   `type` (string): Specifies the type of input (e.g., 'text', 'date').
-   `name` (string): Defines the name of the input element.
-   `label` (string): Provides a text label for the input field. It's used differently based on the variant.
-   `placeholder` (string): Specifies a short hint in the input field.
-   `variant` (string): Determines the style variant of the input field ('unstyled', 'floating').
