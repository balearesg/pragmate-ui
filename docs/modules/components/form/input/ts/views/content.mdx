# Input

El componente `Input` Nos permite agregar elementos input a nuestra aplicacion con la especialidad de poder agregar si
deseamos o necesitamos como atributos un label definido,icono, un disable, contraseña con icono de ocultar y un spinner.

### Import:

<Code>{`import { Input } from 'pragmate-ui/form';`}</Code>

### Component :

<Code>{`<Input type="text" name="name" label="Insert a text..." onChange={handleChange} value={value} />`}</Code>

### Props :

-   `value`: Representa el valor actual del campo de entrada. Si no se proporciona, se establece en una cadena vacía por
    defecto.
-   `errorMessage`: Es el mensaje de error que se muestra cuando el formato del valor es incorrecto. Si no se
    proporciona, se establece en "Formato incorrecto" por defecto.
-   `lengthMessage`: Es el mensaje que indica la cantidad máxima permitida para el valor. Por defecto, se establece en
    "Cantidad máxima: ".
-   `emptyMessage`: Es el mensaje que se muestra cuando el campo de entrada está vacío y es requerido. Por defecto, se
    establece en "Este campo es requerido".
-   `type`: Representa el tipo de entrada del campo. Si no se proporciona, se establece en "text" por defecto.
-   `className`: Es una clase CSS adicional que se agrega al contenedor principal del componente.
-   `icon`: Representa el nombre del icono que se muestra junto al campo de entrada.
-   `loading`: Indica si se debe mostrar un spinner de carga junto al campo de entrada.
-   `password`: Indica si el campo de entrada debe ocultar o mostrar el texto ingresado en forma de contraseña.
-   `required`: Indica si el campo de entrada es obligatorio.
-   `disabled`: Indica si el campo de entrada está deshabilitado para la interacción del usuario.
-   `hasError`: Indica si el campo de entrada tiene un error.
-   `floating`: Indica si la etiqueta flotante debe mostrarse encima del campo de entrada.
