# ButtonGroup

El componente `ButtonGroup` de Pragmate UI sirve para agrupar múltiples botones que comparten un objetivo común en la
interfaz de usuario. Este componente es especialmente útil para crear conjuntos de acciones, ya sean exclusivas, como
las opciones de radio buttons, o no exclusivas, como los filtros de búsqueda.

## Propiedades

A continuación se describen las propiedades disponibles para personalizar del componente:

-   `size`: Define el tamaño de los botones en el grupo, su valor por defecto es `medium` y sus posibles valores son:

    -   `small`
    -   `medium`
    -   `tall`

-   `orientation`: Establece la orientación de los botones, ya sea en fila o columna, por defecto este se mostrara como
    una fila: `row`, sus posibles valores:
    -   `row`,
    -   `column`

## Disabled

**Propiedad: disabledButtonIndexes**

- Descripción: Esta propiedad permite especificar los índices de los botones dentro del componente `ButtonGroup` que se deben deshabilitar.
- Tipo: Array de números.
- Valor predeterminado: `[]` (un array vacío, lo que significa que no se deshabilitarán botones por defecto).

**Uso:**
1. Importa el componente `ButtonGroup` en tu archivo donde deseas utilizarlo.
2. Añade el componente `ButtonGroup` en tu JSX y pasa la propiedad `disabledButtonIndexes` con un array de los índices de los botones que deseas deshabilitar.
3. Cada índice especificado en `disabledButtonIndexes` corresponderá a un botón hijo del `ButtonGroup`.
4. Los botones cuyos índices coincidan con los índices especificados en `disabledButtonIndexes` estarán deshabilitados.

<Code>{implementationDisabled}</Code>

<ButtonGroup orientation="row" disabledIndexes={[1, 3, 5]} size="tall">

	<Button variant="primary">Button 2</Button>

	<Button variant="primary">Button 4</Button>
