# Breadcrumb

El componente `Breadcrumb` facilita la navegación al mostrar la ubicación actual del usuario dentro de la jerarquía de
tu aplicación o sitio web. Cada "miga de pan" es un enlace que apunta a una página o sección diferente, ofreciendo una
manera intuitiva de navegar.

## Instalación

Para incluir el componente `Breadcrumb` en tu proyecto, puedes usar la siguiente declaración de importación:

<Code>{`import { BreadCrumb } from "pragmate-ui/breadcrumb"`}</Code>

## Uso :

<Code>{implementation}</Code>

## Propiedades

-   items: ([string, string]) Es una tupla de arreglos que especifica las "migas de pan". Cada subarreglo contiene dos
    elementos: - Ruta de Navegación: Una cadena de texto que indica la ruta del enlace. Utiliza "backticks" para incluir
    la ruta. - Etiqueta: Una cadena de texto que se mostrará como etiqueta para el enlace.
